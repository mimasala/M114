
# Aufgaben

## RGB
#FF0000 : rot
#00FF00 : grün
#0000FF : blau
#FFFF00 : gelb
#00FFFF : cyan

## CMYK
C:0%, M:100%, Y:100%, K:0% : rot
C:100%, M:0%, Y:100%, K:0% : grün
C:100%, M:100%, Y:0%, K:0% : blau
C:0%, M:0%, Y:100%, K:0% : gelb
C:100%, M:0%, Y:0%, K:0% : cyan

## YCbCr
RGB 255/255/255 in YCbCr -> 1, 0, 0
RGB 0/0/0 ergibt in YCbCr -> 0, 0, 0
Y:1, Cb:0, Cr:0 : weiss
Y:0, Cb:0, Cr:0 : schwarz
Y:0, Cb:1, Cr:0 : rot
